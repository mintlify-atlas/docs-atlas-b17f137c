## Overview

Relio's mapping features provide powerful geospatial visualization and analysis tools for properties, contacts, and territories. Built on Mapbox GL JS, the platform offers interactive maps with multiple basemap styles, drawing tools, and spatial search capabilities.

<Info>
  Maps integrate seamlessly with properties, contacts, and territories to provide location-based insights.
</Info>

## Map Interface

### Basemap Styles

Choose from multiple professionally designed basemap styles:

**Time-of-Day Themes:**
- **Dawn** - Soft morning colors
- **Day** - Bright, high-contrast (default)
- **Dusk** - Warm evening tones
- **Night** - Dark mode optimized

**Specialized Maps:**
- **Satellite** - High-resolution aerial imagery
- **Terrain** - Topographic features and elevation

<Tip>
  Your basemap preference is saved automatically and persists across sessions.
</Tip>

### Map Controls

Built-in controls for map interaction:

- **Zoom Controls** - Zoom in/out with buttons or mouse wheel
- **Compass** - Reset map orientation to north
- **Geolocate** - Center map on your current location
- **Fullscreen** - Expand map to full browser window
- **Draw Tools** - Create custom shapes and areas

## Property Visualization

### Property Markers

Properties are displayed on the map with intelligent clustering:

**Marker Types:**
- Individual property pins at high zoom levels
- Cluster circles showing property count at low zoom
- Custom icons for different property types
- Color-coded markers based on property status

**Interaction:**

## Spatial Search

### Drawing Tools

Create custom search areas using multiple drawing modes:

<AccordionGroup>
  <Accordion title="Polygon Search">
    Draw custom shapes to search properties:
    1. Click "Draw" in the toolbar
    2. Click points on the map to create vertices
    3. Double-click to complete the polygon
    4. All properties within the shape are selected
  </Accordion>
  
  <Accordion title="Radius Search">
    Search within a circular area:
    1. Select "Radius" from draw tools
    2. Click a center point on the map
    3. Drag to set the search radius
    4. Properties within the circle are selected
  </Accordion>
  
  <Accordion title="Rectangle Search">
    Quick rectangular selection:
    1. Hold Shift while dragging on the map
    2. Release to select all properties in the rectangle
  </Accordion>
</AccordionGroup>

## Territory Management

### Creating Territories

Define geographic territories for your team:

```typescript
// Create a new territory with a boundary
const territory = await trpc.territories.create.mutate({
  name: "Downtown District",
  color: "#3b82f6",
  description: "City center coverage area",
  boundary: {
    type: "Polygon",
    coordinates: [[
      [-122.4194, 37.7749],
      [-122.4194, 37.7849],
      [-122.4094, 37.7849],
      [-122.4094, 37.7749],
      [-122.4194, 37.7749]
    ]]
  },
  isExclusive: true
});
```

### Territory Visualization

Territories appear as colored polygons on the map:

- **Fill color** - Translucent territory area
- **Border** - Solid territory boundary
- **Labels** - Territory name and assignees
- **Hover effects** - Highlight on mouse over

<CodeGroup>
```typescript List Territories
const territories = await trpc.territories.list.query();

territories.forEach(t => {
  console.log(t.name, t.assignees.length);
});
```

```typescript GeoJSON Export
// Get territories as GeoJSON for map rendering
const geoJson = await trpc.territories.geoJson.query();

<Source id="territories" type="geojson" data={geoJson}>
  <Layer {...territoryFillLayer} />
  <Layer {...territoryLineLayer} />
</Source>
```
</CodeGroup>

### Territory Coverage

Analyze which properties and contacts fall within territories:

```typescript
const coverage = await trpc.territories.coverage.query({
  territoryIds: ["territory_id_1", "territory_id_2"]
});

coverage.forEach(result => {
  console.log(`${result.territoryName}:`);
  console.log(`  Properties: ${result.propertyCount}`);
  console.log(`  Contacts: ${result.contactCount}`);
  console.log(`  Deals: ${result.dealCount}`);
});
```

### Territory Gaps

Identify properties not covered by any territory:

```typescript
const gaps = await trpc.territories.gaps.query({
  limit: 100
});

console.log(`${gaps.length} properties without territory assignment`);
```

<Warning>
  Territory gap analysis uses PostGIS spatial queries and requires properties to have valid latitude/longitude coordinates.
</Warning>

## Map State Management

The map maintains view state across navigation:

```typescript
interface MapViewState {
  longitude: number;
  latitude: number;
  zoom: number;
  pitch?: number;
  bearing?: number;
}

const DEFAULT_VIEW: MapViewState = {
  longitude: -98.5795,
  latitude: 39.8283,
  zoom: 4
};
```

**Persistence:**
- View state saved to URL parameters
- Restored when returning to map
- Shared via URL for collaboration

## Performance Optimization

### Vector Tiles

For large datasets, properties are rendered using vector tiles:

- Server-side tile generation
- Efficient rendering of 10,000+ properties
- Automatic clustering
- Smooth pan and zoom

Layers are loaded conditionally based on zoom level:

```typescript
// Parcels only at high zoom
minzoom: 14

// Clusters at low zoom
maxzoom: 12

// Individual markers at medium-high zoom

```
