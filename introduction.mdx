## What is Relio?

Relio is a modern CRM built specifically for commercial real estate professionals. It helps you manage relationships, track properties, close deals, and leverage AI to uncover insights across your entire portfolio.

## Architecture

Relio is built as a modern monorepo with the following stack:

- **Frontend**: Next.js 16, React 19, TailwindCSS 4
- **Mobile**: React Native with Expo
- **Backend**: Hono + tRPC for type-safe APIs
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: Better Auth with multi-tenant support
- **Real-time**: Supabase Realtime subscriptions
- **AI**: OpenAI integration for chat and insights
- **Maps**: Mapbox GL for property visualization

## Use Cases

**Property Acquisitions**: Track target properties, research ownership, manage outreach campaigns, and move deals through your pipeline.

**Leasing & Tenant Management**: Maintain relationships with tenants, track lease expirations, and identify expansion opportunities.

**Brokerage**: Manage client relationships, property listings, and deal flow with custom pipelines for different transaction types.

**Development**: Track land acquisitions, maintain relationships with contractors and partners, and manage project timelines.
