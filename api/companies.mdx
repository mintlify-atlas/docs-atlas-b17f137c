## Overview

The companies router manages organizations and businesses in your CRM.

## List Companies

```typescript companies.list
const result = await client.companies.list.query({
  limit: 50,
  search: 'tech',
  type: 'SaaS',
  city: 'San Francisco',
  sort: { field: 'name', direction: 'asc' },
});
```

<ParamField path="limit" type="number" default="50">
  Number of records to return
</ParamField>

<ParamField path="offset" type="number" default="0">
  Number of records to skip
</ParamField>

<ParamField path="search" type="string">
  Search by name or domain
</ParamField>

<ParamField path="type" type="string">
  Filter by company type
</ParamField>

<ParamField path="city" type="string">
  Filter by city (partial match)
</ParamField>

<ParamField path="listId" type="string">
  Filter by list membership
</ParamField>

<ParamField path="tagIds" type="string[]">
  Filter by tags
</ParamField>

<ParamField path="sort" type="object">
  Sort configuration
  <Expandable title="Sort fields">
    Available fields: `name`, `type`, `city`, `createdAt`
  </Expandable>
</ParamField>

<ResponseField name="items" type="array" required>
  Array of company objects
</ResponseField>

<ResponseField name="total" type="number">
  Total count
</ResponseField>

<ResponseField name="nextCursor" type="string">
  Pagination cursor
</ResponseField>

## Update Company

```typescript companies.update

  id: 'company_id',
  employeeRange: '51-200',
  fundingRaised: 5000000,
});
```

<ParamField path="id" type="string" required>
  Company ID
</ParamField>

## Enrich Company

```typescript companies.enrich
const result = await client.companies.enrich.mutate({
  id: 'company_id',
});
```

Enriches company data from Firecrawl using the company's domain. Falls back to Exa if Firecrawl fails.

<Note>
  Requires FIRECRAWL_API_KEY. This is a heavy operation.
</Note>

<ResponseField name="company" type="object">
  Updated company object
</ResponseField>

<ResponseField name="enrichedFields" type="string[]">
  Fields that were populated
</ResponseField>

<ResponseField name="source" type="enum">
  Data source: `firecrawl` or `exa`
</ResponseField>

<ResponseField name="exaResults" type="array">
  Search results when using Exa fallback
</ResponseField>

### Enrichment Data

Firecrawl enrichment populates:
- Description (AI-generated summary)
- Contact information (email, phone)
- Logo URL
- Social profiles (LinkedIn, Twitter, Facebook, Instagram)
- Categories and industry tags
- Employee range estimate
- Revenue estimate (ARR)
- Funding amount
- Foundation date

## Aggregations

```typescript companies.aggregations
const stats = await client.companies.aggregations.query({
  aggregations: [
    { type: 'count' },
    { type: 'group', column: 'type' },
    { type: 'group', column: 'city' },
    { type: 'sum', column: 'fundingRaised' },
  ],
});
```

Supported columns:
- `name`, `type`, `city`, `twitterFollowerCount`, `fundingRaised`, `createdAt`
- `tags` (junction table)
